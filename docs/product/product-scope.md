# Product Scope

## Vision

Cozy Cabin is a cozy virtual study space designed to help people focus, build sustainable study habits, and eventually connect with others through meaningful interactions.

The long-term vision is to combine productivity, relaxation, and community into a single online experience.

---

## Guiding Principles

This project follows a few simple principles:

* Build a real product, not just a portfolio demo.
* Keep the first version as small as possible.
* Finish a complete feature before starting the next one.
* Choose maintainability over unnecessary complexity.

---

## Milestone 1 (MVP)

The goal of the first milestone is to prove that the core full-stack architecture works from end to end.

A user should be able to:

* Create an account
* Sign in
* Access a personal dashboard
* Start and complete a Pomodoro session
* View previously completed study sessions

Completing this milestone means the application already has a complete flow from frontend to backend to database.

---

## Future Milestones

Future versions of Cozy Cabin may include:

* Pen pal system
* Letters and inbox
* Ambient music
* Cozy room customization
* AI-assisted productivity features

These features are intentionally postponed until the core product is stable.

---

## Out of Scope (Milestone 1)

The following features will **not** be included in the first milestone:

* Real-time chat
* Multiplayer study rooms
* Video or voice calls
* AI features
* Public communities
* Mobile application

---

## Success Criteria

Milestone 1 is considered complete when:

* A new user can successfully register and sign in.
* A user can complete a Pomodoro session.
* Study sessions are stored in the database.
* Study history is displayed on the user's dashboard.
* The application is deployed online and accessible through a public URL.
